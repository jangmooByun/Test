컴퓨터공학과에서는 컴퓨터 시스템 및 소프트웨어 개발 능력과 현장 실무능력을 지닌 전문 엔지니어를 양성하고 있습니다.학과 특성을 지향하는 학술 동아리에 모든 학년이 함께 활동하고 있고, IT관련 전문 지식을 습득하고 관련 분야의 국제공인자격증 등을 취득하고 있습니다.Cisco와 Java 학술동아리는 매년 CCNA(네트워크 국제공인자격증)와 SCJP(자바프로그래머 국제공인자격증)를 취득하고 있으며, CNAP(Cisco Networking Academy Program) 프로그램을 운영하고 있습니다.소프트웨어 개발, 서버시스템 모바일 등의 프로그램을 산학협력과 실무중심으로 교육하여 매우 높은 취업률달성에 노력하고 있습니다. 실무경력이 풍부한 교수진들이 최신IT시설을 갖춘 다양한 교육시설과 최고 수준의 연구 환경을 바탕으로 현장 실습형 교육을 제공합니다.

멀티미디어기초 Multimedia Basics
웹 디자인에 필요한 이미지 및 그래픽 편집 툴의 활용 기법을 습득하고 조형, 색채, 구성 등이 조화된 창의적인 웹 컨텐츠 제작 능력을 배양한다.

이산구조 Discrete Mathematics
컴퓨터 분야와 관련된 수학적 이론을 제시하는 과목으로써, 교육 내용은 논리 개념과 증명, 집합과 상호관계, 그래프와 트리 구조의 개념과 응용, 알고리즘의 개념과 분석 등이다.

시스템활용 Practical use of UNIX
멀티테스킹이 가능하고 다양한 사용자가 공유할 수 있는 UNIX시스템의 기본 및 일반텍스트파일의 저장, 계층적인 파일시스템구조, 장치와 프로세스 간 통신을 파일로 취급하는 다양한 콘셉트의 특징을 이해하고 인터넷환경과 네트워크에 응용할 수 있는 지식을 습득한다.

자료구조 Data Structures

데이터베이스 Databases
대량의 데이터를 관리할 수 있는 데이터베이스에 대한 기본 이론을 학습하고 효율적인 데이터베이스의 설계 및 정보처리 기술을 습득한다.

객체지향언어 Object-oriented Language
C++과 같은 객체 지향적 언어를 통하여 객체지향 프로그래밍 능력을 배양한다.

서버구축 및 관리 Server Management
리눅스를 기반으로 하는 서버 시스템의 구축, 운영에 필요한 기초 지식 및 활용 능력을 배양하고 인터넷 환경에서 효율적으로 리눅스 기반 서버 시스템을 관리할 수 있는 능력을 배양한다.

논리회로 Logic Circuits
컴퓨터 등 디지털 정보처리 시스템을 이해하기 위한 기본 개념 및 원리를 이해한다.

멀티미디어응용 Multimedia Applications
웹 디자인에 필요한 애니메이션 편집 툴의 활용 기법을 습득하고 이미지, 애니메이션, 액션스크립트 등이 조화된 창의적이고 동적인 웹 컨텐츠 제작 능력을 배양한다.

파일처리 File Processing

운영체제 Operating Systems

데이터베이스응용 Applications of database management system
데이터베이스 연구에 기본이 되는 필수이론을 습득한다. 이 과목에서는 관계형 데이터베이스 이론, 일반 종속이론과 회복기법 등을 다루며, 객체지향 데이터베이스와 Web과의 응용방법 등에 대하여 실습을 한다.

정보통신 Information Communication
정보화 사회의 기반인 정보에 대한 기본 개념 및 통신 프로토콜을 학습하여 정보통신에 대한 실무적응력을 배양한다.

웹서버 프로그래밍 Web Programming

컴퓨터구조 Computer Architecture

소프트웨어공학 Software Engineering

컴퓨터네트워크 Computer networks
인터네트워킹 및 근거리 통신망에 대한 원리 및 라우팅 통신 프로토콜 등을 학습하고 네트워크설계, 운영, 관리할 수 있는 능력을 배양한다.

알고리즘 Design and Analysis of Algorithm

컴퓨터그래픽스 Computer Graphics

네트워크보안 Network Security
